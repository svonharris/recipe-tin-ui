# Color

Recipe Tin UI is built on a structured color system using CSS custom properties (design tokens). Colors are organized into brand palettes, neutral palettes, and semantic palettes — each with a scale from `50` (lightest) to `950` (darkest). These are the core colors that define the Recipe Tin UI visual identity.

## Usage

All colors are defined as CSS custom properties in [tokens.css](src/styles/tokens.css) and are available globally. Always reference colors via their token rather than hardcoding hex values directly.

```css
/* Do */
color: var(--color-primary-500);

/* Don't */
color: #26ae60;
```
