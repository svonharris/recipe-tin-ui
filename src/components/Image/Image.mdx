# Image

A wrapper around the native `<img>` element that adds support for border radius tokens. Width and height are passed as strings, accepting any valid CSS value.

## Guidelines

- Always provide a descriptive `altText` — this is used as the `alt` attribute and is required for accessibility.
- Use border radius tokens from the design system rather than hardcoded values (e.g. `var(--border-radius-lg)`).
- Set `width` and `height` explicitly when the image dimensions are known, to prevent layout shift while the image loads.
