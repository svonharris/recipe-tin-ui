# Button

Allows the user to perform an action.

## Guidelines

- Do not use a Button to link to another page. Use a [Standalone Link](/story/components-link--navigation) or a [Call to Action Link](/story/components-link--arrow) instead, or an [inline Link](/story/components-link--default) in the running text.
- Describe the Button’s function through a short label.
