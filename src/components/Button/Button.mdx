# Button

Allows the user to perform an action.

## Guidelines

- Do not use a Button to link to another page. Use a Standalone Link or an inline Link in the running text instead.
- Describe the Button's function through a short, clear label.
- Use only one primary button per view. Multiple primary buttons compete for attention and reduce clarity.
- Pair a primary button with a ghost button when two actions are needed.
