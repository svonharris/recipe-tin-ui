# Container

A layout component that wraps content in a configurable HTML element with controllable padding on each side.

## Guidelines

- Use Container to add consistent horizontal padding around page-level or section-level content.
- Use the `paddingLeft` and `paddingRight` props to override the default `16px` horizontal padding.
- Use `paddingTop` and `paddingBottom` to add vertical spacing when needed.
- Use the `htmlElement` prop to render the appropriate semantic HTML element (`main`, `section`, `article`, etc.).
