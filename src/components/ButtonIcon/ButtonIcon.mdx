# Icon Button

An icon-only button that allows the user to perform an action. Optionally supports a label beneath the icon.

## Guidelines

- Always provide a `title` prop for accessibility — this is used as the button's tooltip and screen reader label.
- Use an icon button when space is limited and the icon clearly communicates the action without a text label.
- Add a `children` label beneath the icon when the action may not be immediately obvious from the icon alone.
- Use only one primary icon button per group of related actions.
