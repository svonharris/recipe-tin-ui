# NavBar

A navigation bar that provides links to key areas of the application.

## Guidelines

- Provide a clear, concise `name` for each nav item so users can quickly identify destinations.
- Use a recognisable icon alongside each label to reinforce meaning at a glance.
- Keep the number of nav items small — typically three to five — to avoid overwhelming the user.
- Ensure each `href` points to a distinct, meaningful route.
