# Recipe Gallery

Renders a collection of RecipeCards from an array of recipe data.

## Guidelines

- Pass an array of recipe objects via the `recipes` prop, each containing a `label`, `images.REGULAR.url`, and `totalTime`.
- The gallery renders each recipe as a RecipeCard in the order provided.
- A `totalTime` of `0` will display "n/a" in the RecipeCard's duration field.

## Usage

`RecipeGallery` is display-only — it has no knowledge of where the data comes from. Pair it with a data source in the parent.

```tsx
import { useEdamamSearch } from "./hooks/useEdamamSearch"; // your app's hook
import { FormSearch, RecipeGallery } from "recipe-tin-ui";

const [recipes, setRecipes] = useState<any[]>([]);
const { search } = useEdamamSearch(setRecipes);

<FormSearch onSubmit={search} />
<RecipeGallery recipes={recipes} />
```

`FormSearch` calls `onSubmit` with the query string. Pass your search handler directly — `RecipeGallery` renders whatever is in `recipes`.
