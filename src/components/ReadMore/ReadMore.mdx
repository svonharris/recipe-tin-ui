# Read More

Truncates long text at 150 characters and lets the user expand or collapse it.

## Guidelines

- Use ReadMore when text content may be too long to display in full by default.
- Text shorter than 150 characters is displayed in full with no toggle shown.
- Use the `className` prop to apply a typography class consistent with surrounding text.
