# Stack

A layout component that arranges children vertically in a column.

## Guidelines

- Use Stack to group elements that should flow from top to bottom.
- Use the `gap` prop to control the space between children.
- Use the `align` prop to control horizontal alignment of children within the column.
- Use the `justify` prop to distribute children across the available vertical space.
