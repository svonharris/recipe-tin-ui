# RecipeRecommend

Fetches and surfaces a random selection of recipe recommendations from the Edamam API.

## Guidelines

- `RecipeRecommend` is a renderless component — it returns no UI of its own.
- Use the `onResults` callback to receive the fetched recipes and pass them to a display component such as `RecipeGallery`.
- Use `limit` to cap the number of recipes returned. If omitted, all results from the API are passed to `onResults`.
- Recipes are fetched once on mount using a randomly selected food category.
